## ✨ What's New? The Great Pivot!
EZ-Keys has evolved from a simple chord search tool into a **Global Chord Library Browser**:
- 📚 **Key-Based Navigation**: Select a Global Key and easily step through all its diatonic chords using left/right arrows. No more typing!
- 🔢 **Nashville Number System**: Toggle seamlessly between traditional Letters and the Nashville Number System globally across all views.
- ⏱️ **Setlist Sequencer**: Create setlists with actual progressions (e.g. `1 5 6m 4`). Hit **Play** to watch the chords animate across a real-time, 2-octave mini keyboard based on your BPM and Time Signature.

## 🚀 Core Features

- **Chord Explorer**: See what the audience hears versus what you physically play.
- **Voicing Explorer**: Generate practical two-hand voicings (shell, rootless, gospel) instantly mapped to your reference key.
- **Setlist Mode with Sequencer**: Play along with a built-in visual sequencer that guides you through every bar of your songs.
- **Progression Translator**: Test out chord progressions and watch them translate perfectly to your transpose setting.
- **Scale Explorer**: Visualize scales and modes.
- **Modulation Assistant**: Calculate smooth key transitions.
